## Quick Command Reference

### List Operations
```bash
# Add to list
operation: add_item
required: list_name, item_text

# Show list
operation: list_items
required: list_name

# Show all lists
operation: get_lists
```

### Task Operations
```bash
# Create task
operation: create_task
required: title
optional: description, due_at, priority

# List tasks
operation: list_tasks
optional: completed (true/false filter)

# Complete task
operation: complete_task
required: task_id
```

### Note Operations
```python
# Save note
MemoryItem(
    source=MemorySource.CAPTURE,
    title="short summary",
    content="full text",
    # Optional enhancements:
    people=["names"],
    tags=["categories"],
    user_id="id",
    chat_id="id"
)
```
